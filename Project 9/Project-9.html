Introduction
The purpose of this report is to use empirical data to see if the economy of the Ivory
Coast can be modeled with the Keynesian Theory of Consumption. If the economy
of the Ivory Coast does conform to this model, then its real disposable income per
capita should be directly proportional to its real private consumption per capita.
Deriving the Necessary Variables
The variables for this report are somewhat difficult to record by organizations due to
the large scale and ever-changing nature of economic data, so most data sets
exclude these as they can be derived from others. Said derivations will be conducted
for the purpose of this inquiry. In this report, the household disposable income was
found by using the equation Y = Yd T where Y is output/income, Yd is household
disposable income and T is income taxes. Both Y and T were reported as nominal
values in the local currency Western African CFA Franc, thus the value of Yd is the
same. Yd was adjusted with the CPI index base year 2010 to obtain real values, then
divided by population to get per capita value. World Bank has a data set on final
consumption in nominal CFA Franc which is private consumption plus government
final consumption. Government final consumption was then found and subtracted
from final consumption to get private consumption. This nominal value was then
adjusted with the CPI to get real private consumption, then divided population for the
per capita value. This raw data is summarized in detail in Table 4 and Table 5 of the
appendix, with Table 3 summarizing the final calculated variables.
Method of Data Analysis
The real disposable income per capita and real private consumption per capita, both
in CFA Franc, were regressed using Excels data regression function in the Analysis
Toolpak Add-in to find a straight line of best fit. Its equation is the consumption
function of the Ivorian economy (see Graph 1 and Table 3 in appendix). This
regression line slope is equal the marginal propensity to consume, and its intercept
is equal to autonomous spending.
Analysis of Consumption Function from Regression Model
The slopes value is 0.5463 which conforms to the Keynesian consumption model
that has the marginal propensity to consume to be a value between 0 and 1. It implies
Ivorians spend approximately 0.55 cents of every dollar they earn and save the
remaining 0.45 cents. The intercept is equal to 85,334 suggesting that Ivorians
autonomous consumption is 85,334 CFA Franc, which is approximately 191

Analysis of the Regression Accuracy
The coefficient of determination (R2) is 0.907 (see Table 2), meaning that 90.7% of the
variation in real private consumption can be explained by the variation in real
disposable income for the Ivorian economy. The correlation coefficient (Multiple R) is
0.952 (see Table 2), meaning the strength of the linear relationship is 95.2%. This
suggests the line of best fit is accurate given the inputted variables, and thus is a
good representation of the true Ivorian consumption function.
The intercept standard error is quite high relative to the intercept value (see Table 1),
but the slope standard error is quite low relative to the slope value. The true value of
the slope must lie within a small range around the value calculated whereas the true
value of the intercept has a large range which suggests mild inaccuracy of the true
value. Both p-values are much less than 0.01 (see Table 1) which suggest our results
are statistically significant.

Conclusion
The Keynesian Theory of Consumption models the Ivorian economy well with regards
to its consumption. In this report it was shown that just over half of what Ivorians
earns is spent. This is a relatively high amount, but conforms to the trend of
developing nations having relatively high marginal propensities to consume. People
in developing nations tend to not have many of their immediate basic needs met, and
must devote the majority of their income to subsistence consumption.
